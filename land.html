Comandos para comitar no git pelo terminal.

  • git --version
  • git status
  • git init
  • git add <arquivo>
  • git add . :todos arquivos
  • git rm --cached <arquivo>
  • git commit -m "mensangem"
  • git log

Para ignorar algum arquivo que nao queira atualizar com o commit,
basta criar um arquivo na pasta chamado .gitignore e adicionar
os arquivos que queira ignorar

Criando uma branch

  • git branch <nome da brach> ...cria uma branch
  • git branch ...consulta as branchs
  • git checkout <nome da brach> ...entra na branch
  • git merge <nome da brach> ...adiciona os arquivos da branch selecionada
